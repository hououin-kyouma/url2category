How to manually add a wireless network in Windows 7 and Vista

Was this article helpful?   Yes     No | 71 people found this helpful in last 30 days

If a wireless network has been configured not to broadcast the SSID, it will not be detected and will not appear in the list of available wireless networks. However, if the details of the wireless network are known, it is possible to connect to it by manually by creating a wireless network profile on the computer.


To manually add a wireless network in Windows 7 and Vista:

  1. In the Control Panel, go to the Network and Sharing Center and click Manage Wireless Networks.

  2. Click Add and click Manually create a network profile.

  3. Manually add the wireless settings for the network that you want to connect to (Network name, Security type, Encryption type, and Security key/passphrase).
    Select Start this connection automatically and Connect even if the network is not broadcasting. Click Next.

  4. Make sure the wireless network name is in the list of preferred networks.

  5. The computer should now connect automatically to the wireless network.
    To check if it has successfully connected to the wireless network and obtained an IP address, check the IP address of the Wireless Network Connection, using the command line command ipconfig.

  6. Ping the Default Gateway. If that works, the computer is successfully communicating with the local network.

    Next, ping the IP address 4.2.2.2 . If this is successful, the computer is communicating successfully with the internet.

    Next, ping www.google.com, or some other internet host that is known to respond to ping. If this is successful, DNS is working ok and the computer will be able to browse the internet.
