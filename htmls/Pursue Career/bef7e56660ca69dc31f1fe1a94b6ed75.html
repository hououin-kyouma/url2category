Course Admission Requirements

    How to Apply

    Contact campus for application and enrolment information.

    Note: You must be an Australian Resident to apply for this course. For International Students please visit International Courses.

    Employment Outcomes

    This skill set is for persons who are able to work as application programmers and software developers.

    Note: Completion of a TAFE SA course does not guarantee an employment outcome. Formal requirements (eg licensing, professional registration), may apply to some occupations.

    Skills Covered

    Application Development Specialist Skill Set (ICTSS00031)
    5 units:

    • ICTPRG501 Apply advanced object-oriented language skills 30 hrs
    • ICTPRG505 Build advanced user interface 60hrs
    • ICTPRG506 Design application architecture 40 hrs
    • ICTPRG523 Apply advanced programming skills in another language 80 hrs
    • ICTPRG507 Implement security for applications 50hrs:
      • Pre-requisites:
        • ICTPRG301 Apply introductory programming techniques
        • ICTWEB301 Create a simple mark-up language document
        • ICTPRG402 Apply query language | 4DBB
        • ICTPRG410 Build a user interface
        • ICTPRG414 Apply introductory programming skills in another language
        • ICTPRG406 Apply introductory object-oriented language skills
        • ICTPRG405 Automate processes
        • ICTPRG415 Apply skills in object-oriented design
        • ICTPRG419 Analyse software requirements.

    Note: Students who successfully complete one or more units of competency will receive a Statement of Attainment.
